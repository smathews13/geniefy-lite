### LOW severity
- **`overall` rollup uses the JS `??` short-circuit, not the literal `if-else` of the server — semantically equivalent but worth a note.** `ReviewView.tsx:74-75` writes `tableConf != null && colMean != null ? 0.5*tableConf + 0.5*colMean : tableConf ?? colMean`. The server (`api.py:110-113`) uses an explicit `if table is not None and col_mean is not None … else table_conf if table_conf is not None else col_mean`. Because `tableConf`/`colMean` are `number | null` (never `undefined`/`0`-as-absent — `0` is a legitimate confidence and `??` only coalesces `null`/`undefined`), the two are equivalent for every input: both-present → weighted; one-present → that side; both-null → `null`. A `tableConf` of exactly `0` flows through `0.5*0 + 0.5*colMean` correctly (not short-circuited, since `0 != null` is true). No defect; flagged LOW only because the divergent expression shape could mislead a future reader into thinking `??` treats `0` as absent — it does not. No action required.
