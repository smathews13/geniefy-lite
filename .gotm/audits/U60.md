- **`__table__` sentinel + `encodeURIComponent`.** ReviewView passes `TABLE_TARGET` (`'__table__'`, types.ts:81) as the table card's `target` (`ReviewView.tsx:20`) and `column_name` for column cards (`:23`); DraftCard threads `target` unchanged into `useReview`, which `encodeURIComponent`s it before the URL (hooks.ts:71). Correct for both the sentinel and arbitrary column names.
- **Null-guards on optional draft fields.** `current_comment &&` (DraftCard.tsx:68), `proposed_comment ?? '—'` (`:98`) and `??
