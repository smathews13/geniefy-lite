- Apply derives the table key as `cat, sch, tbl = _split(state.target)` ‚Üí `f"{cat}.{sch}.{tbl}"`, where `_split` (`apply.py:46`) is `[p.strip().strip("`") for p in str(target).split(".")]` (`apply.py:169` joins the 3 parts).

Column `match_key` is the raw `target_name` ‚Äî unchanged, and apply also uses the raw column name (`apply.py:169`), so columns were and remain consistent.
- Empirically confirmed equivalent across clean (`samples.tpch.orders`), whitespace-padded (`  samples . tpch . orders `), backtick-quoted (`` `samples`.`tpch`.`orders` ``), and a second clean FQN ‚Äî every case `apply == approve` (byte-identical).
