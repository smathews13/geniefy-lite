## Check 1 — new inputs distinguish weighted from mean
The fixed test (`tests/test_judge.py:92–96`) feeds `{completeness=1, specificity=0, grounding=0, template_conformance=0}`:
- weighted = `1*0.30 + 0*0.30 + 0*0.25 + 0*0.15` = **0.30**
- plain mean = `(1+0+0+0)/4` = **0.25**

The two values differ (0.30 ≠ 0.25), unlike the prior `{1,1,0,1}` case (0.75 = 0.75). The test asserts both `abs(overall - 0.30) < 1e-9` (the template-weighted value) **and** `abs(overall - 0.25) > 1e-3` (explicitly NOT the plain mean). The default weights are confirmed `{0.30, 0.30, 0.25, 0.15}`, so 0.30 is the correct weighted result. **PASS.**
